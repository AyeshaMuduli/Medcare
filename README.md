# MedCare

##  Project Overview
MedCare is an Android medicine delivery application developed using Kotlin and Jetpack Compose. The app allows users to browse medicines, search by category, add medicines to the cart, place orders, and view their order history.

##  Features
- User Registration and Login
- User Profile
- Medicine Categories
- Search Medicines
- Medicine Details
- Add to Cart
- Checkout
- Order History
- SQLite Database for User Data
- Modern Jetpack Compose UI

##  Technologies Used
- Kotlin
- Jetpack Compose
- Material 3
- SQLite Database
- Android Studio
- Git & GitHub
